pip install requests
pip install beautifulsoup4
-------------------------------------------------------------------------------------------------------or
Installing requirements.txt

1. Open a command prompt or terminal.

2. Navigate to the directory where the requirements.txt file is located.

3. Run the following command to install the dependencies:

4. pip install -r requirements.txt

This command will instruct pip to read the requirements.txt file and install all the listed dependencies.

Note: Make sure you have pip installed and added to your system's PATH environment variable.
